AutoPort Manager

Professional Car Inventory Management

React Application

This is a React-based car inventory management system. To run the full application, you need Node.js installed.

Features Included:

  • • Beautiful Login/Signup System
  • • Dashboard with Statistics Cards
  • • Car Inventory Table with Search
  • • Add/Edit Car Forms
  • • Responsive Design
  • • Professional UI/UX

To Run the App:

1. Install Node.js

2. Run: npm install

3. Run: npm start
